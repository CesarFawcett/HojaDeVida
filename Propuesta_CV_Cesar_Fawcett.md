## 🚀 Mi Portafolio
*(Aquí pondrás el desglose de los proyectos para asombrar a los evaluadores técnicos)*

1. **Plataforma Web de Duelos y Motor de Cartas (GameAppCeLu)**
   *   **Descripción:** Un ecosistema inmersivo de juego de cartas vía web soportado por un backend persistente, sistema dinámico de compras y sistema de fases y reglas estrictas.
   *   **Logros Técnicos:** Desarrollo de una tienda web con barreras impuestas por la progresión del usuario (derrotas de *bosses*), manejo dinámico de memoria y *arrays* limitados (ej. mecánicas de sobreescritura de cartas-trampa), optimizando el UX en dispositivos móviles gracias a la detección de eventos táctiles dinámicos.

2. **Sistema de Streaming Remoto (Android-PC Connectivity)**
   *   **Descripción:** Aplicación cliente (PC) – servidor (Android) diseñada para transmitir la pantalla del móvil bajo demanda y baja latencia.
   *   **Logros Técnicos:** Implementación de interacciones directas con el Sistema Operativo, manejo exitoso del `Foreground Services API` en Android, manejo de interfaces asíncronas y resolución de dependencias de compilación para la entrega final del `.apk`.

3. **Arquitectura E-commerce en Java (TiendaApp / API Turismo)**
   *   **Descripción:** Suite de aplicaciones corporativas diseñadas en Java Spring Boot para operar la gestión administrativa, ventas virtuales y lógica comercial en esquemas del lado del servidor.
   *   **Logros Técnicos:** Diseño completo de una arquitectura escalable, modelado relacional de datos a gran escala, implementando controladores, inyección de dependencias e inicialización segura de contextos en *runtime*.

4. **Simulador de Neuroevolución (Dino Game AI)**
   *   **Descripción:** Motor de juego en 2D enlazado con un agente de IA que domina el escenario de forma completamente autónoma.
   *   **Logros Técnicos:** Construcción y entrenamiento desde cero de una *Neural Network* que incorpora mecánicas genéticas y cálculo de redes de función. Incorporé también un depurador visual en alta fidelidad donde se ven las inferencias neuronales mientras la IA "mira y procesa" su entorno en tiempo real. 

5. **El Proyecto "Geometric Pacman"**
   *   **Descripción:** Conceptualización de un motor de laberintos de lógica pura fundamentado estrictamente en geometrías puras y sin dependencias o assets externos, bajo lineamientos rígidos de desarrollo (firma César Fawcett).
   *   **Logros Técnicos:** Resolución de recorridos matriciales complejos e inteligencia artificial de persecución.
